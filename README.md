# Cucumber BDD Automation Project

## 📌 Project Description
This project demonstrates BDD framework using Cucumber, Selenium, and Java.

## 🛠️ Technologies Used
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG / JUnit
- Maven

## 📂 Project Structure
- Feature files: src/test/resources
- Step Definitions: src/test/java
- Test Runner: runner package

## ▶️ How to Run
1. Clone the repo
2. Run TestRunner.java

## ✅ Scenarios Covered
- Valid Login
- Invalid Login (Scenario Outline)
